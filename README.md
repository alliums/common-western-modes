### common-western-modes
A simple selection of the absolute most fundamental Western musical modes.

###contents
the 7 diatonic modes:
1. Lydian
2. Ionian
3. Mixolydian
4. Dorian
5. Aeolian
6. Phrygian
7. Locrian

8. The pythagorean pentatonic scale
